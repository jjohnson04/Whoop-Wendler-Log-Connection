# Privacy Policy

**Last updated: April 22, 2026**

## Overview

This application is a personal-use dashboard that displays WHOOP recovery data alongside Wendler 5/3/1 lift history. It is not a commercial product and is not intended for use by third parties.

## Data Collection

This application does not collect, store, or transmit any personal data to external servers. All data processed by this application remains local to your browser.

## WHOOP Data

This application connects to the WHOOP API using OAuth 2.0 to retrieve your personal recovery, strain, and sleep data. Your WHOOP credentials are never stored by this application. Access tokens are held temporarily in your browser session and are not persisted beyond that session.

## Fitness Data

Workout data imported via CSV upload is processed locally in your browser. No CSV data is uploaded to or stored on any server.

## Third-Party Services

This application uses the WHOOP API. Your use of WHOOP data is subject to [WHOOP's Privacy Policy](https://www.whoop.com/us/en/privacy-policy/).

## Contact

This is a personal project. For questions, contact the repository owner via GitHub.
